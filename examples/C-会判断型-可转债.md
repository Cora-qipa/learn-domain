# 走查样例 · 可转债 / 模式②(会判断)

> 展示模式②(会判断型)+ 慢变金融领域的高风险处理。略去部分轮次。

---

**用户:** /learn-domain 可转债,我在考虑要不要买点

**Skill(阶段0·定终点 + 高风险免责):**
先说一句:我能帮你把可转债这个东西的底层逻辑讲透、建立判断框架,但**这不构成投资建议**,真要下单前请结合自己的风险承受能力,必要时咨询持牌人士。

你的目标很清楚——是 **②会判断型**:不是要研究透每个公式,而是要能判断"这东西适不适合我、该看哪些指标"。那我们就奔着"建立一套判断框架"去,讲到你能自己看懂一只可转债的关键维度为止。

---

**Skill(阶段1·测绘地形):**
可转债的判断地图:

```
 ① 地基:它到底是什么 —— "债 + 一张能换股的票"
 ② 核心三维度(判断就靠这三个):
     转股价值 / 纯债价值 / 溢价率   ← 80/20 全在这
 ③ 进阶:强赎条款、下修条款(影响风险收益,想深入再说)
 两种玩家视角:当债来配置(求稳) vs 博弈转股(求弹性)
```

判断一只可转债值不值,本质就是看 ② 那三个维度。我们重点打通它。

---

**Skill(阶段2·诊断):**
两个小问题校准下:
1. 股票和债券的区别你清楚吗?
2. "下有保底、上不封顶"这句话,你能猜到它想说可转债的什么特点吗?

**用户:** 股债区别知道,一个是股权一个是借钱。第二句不太懂。

**Skill:**(画像:有股债基础→可直接用它搭桥;不懂"下有保底"→正是可转债的内核,从这切入。)

---

**Skill(阶段3·讲解,以"可转债是什么"为例):**

**【点】** 一句话:可转债 = **一张债券,但额外附赠了"在约定价格把它换成股票"的权利**。
类比:像你借钱给一家公司(这是债,到期还本付息,**保底**),但合同里多了一条——如果这公司股价涨得好,你可以选择不要利息了,直接换成它的股票赚差价(**上不封顶**)。
(⚠️ 简化:这个"换"有条件和时间窗口,不是随时随意换;"保底"也以公司不违约为前提。类比在"无条件保本"这点上不成立。)

**【线-纵·它怎么来的】**
公司为什么发这种又是债又是股的东西?早年中小公司想融资,直接发债利息太高、直接增发股票又怕股价被砸。于是设计出可转债:给投资人一个"保底+期权"的甜头,公司就能用很低的利息借到钱——投资人赌的是未来转股的上涨空间。一个两头都想占便宜的产物。

**【线-横·跟谁区分】**
跟纯债比:可转债利息低,但多了股价上涨的弹性。
跟股票比:可转债跌有债底兜着,跌幅通常小于正股,但涨的时候有溢价拖累、不如直接买股票猛。
**所以它的生态位是"想要股票的弹性,又怕股票的回撤"的人。** 这正是你要判断的:你是这种人吗?

清楚了吗?清楚我们就看判断它的那三个核心维度。

---

**Skill(阶段5·沉淀):**
- **判断框架**:任何一只可转债,先看转股价值、纯债价值、溢价率这三个;再看强赎/下修条款。
- **术语表**:溢价率(现价比转股价值贵多少)/ 纯债价值(把它当普通债券值多少,即债底)/ 强赎(公司在股价高时强制赎回,逼你转股)。
- **再次提醒**:框架是帮你看懂,不是替你决定。真要买,结合自己仓位和风险,必要时咨询专业人士。
