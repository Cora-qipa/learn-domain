# 触发矩阵 / Trigger Matrix

> 验证 SKILL.md 的 `description` 触发准确:该触发的全中,不该触发的不误触。
> 跑法:把"输入"作为新会话首条消息,看 learn-domain 是否被激活。

## 应触发(should trigger)

| 输入 | 命中的触发词 |
|---|---|
| `/learn-domain 期权` | 显式调用 |
| 带我入门期权交易 | "入门 X" |
| 想快速了解一下 RAG 是什么 | "快速了解 X" / "X 是什么" |
| 这篇讲扩散模型的文章我完全看不懂 | "看不懂 X 文章" |
| 我想转行做数据分析,从零开始 | "想转行 X" |
| 可转债到底是个什么东西 | "X 是什么意思" |
| 帮我扫盲一下量子计算 | "扫盲" |
| 最近老看到"价值投资",能给我讲讲这个领域吗 | "讲讲这个领域" / 入门意图 |

## 不应触发(should NOT trigger)

| 输入 | 为什么不该触发 | 期望走向 |
|---|---|---|
| 帮我写个快速排序算法 | 是写代码,不是入门领域 | 直接写代码 |
| 这段 Python 报错怎么修 | 调试任务 | 直接调试 |
| 把这句话翻译成英文 | 翻译任务 | 直接翻译 |
| 解释一下这个函数在干嘛 | 读单段代码,非入门领域 | 直接解释 |
| 期权现在的行情怎么样 | 查实时行情,非系统入门 | 联网查行情 |
| 总结一下这篇文章 | 摘要任务,非入门 | 直接摘要 |

## 边界用例(judgment calls,记录预期)

| 输入 | 预期 | 理由 |
|---|---|---|
| 这个概念是什么意思?(单个孤立术语,无入门意图) | 不触发,直接答 | 用户要一句话答案,不是入门一个领域 |
| 给我讲讲机器学习(很宽的领域) | 触发,但阶段0必须先锁终点收窄 | 宽领域更需要终点模式定边界 |

## 通过标准
- "应触发"全部命中。
- "不应触发"零误触。
- 边界用例行为符合"预期"列。
